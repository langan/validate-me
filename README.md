# Validate::Me

Rack Middleware that validates HTML, for use with middleman

## Installation

Add this line to your application's Gemfile:

    gem 'validate-me'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install validate-me

## Usage

Add the following to your middleman config.rb file

    require 'validate-me'
    use Rack::Validate
